🤖 CodeChef Automation + LEADGAME Solution


🚀 Project Overview

This project combines automation using Selenium (Python) and a C++ competitive programming solution to streamline the process of logging into CodeChef, navigating to a problem, and submitting code automatically.

It demonstrates skills in:

Web automation
Competitive programming
Workflow optimization

🧩 Project Components


1️⃣ Selenium Automation Script (Python)

Automates the following steps:

Opens CodeChef website
Logs in using user credentials
Navigates to a specific problem
Uploads solution code
Selects programming language
Submits the solution


2️⃣ C++ Solution (LEADGAME Problem)


🧠 Problem Logic
The problem tracks cumulative scores of two players across rounds and determines:
Who had the maximum lead
The value of that lead



⚙️ Approach
Store scores using pair<int, int>
Maintain cumulative sums
Calculate lead at each step
Track maximum lead and winner


🛠️ Technologies Used
Python (Selenium)
C++ (STL)
Web Automation
Competitive Programming


⚡ Features
Automated login and submission process
Secure password input using getpass
Efficient cumulative sum logic
Optimized I/O for competitive programming


⚠️ Important Notes
Replace 'C:\webdrivers\chromedriver.exe' with your actual path
Add your CodeChef username manually in the script
Make sure solution.cpp exists in the same directory
Selenium methods like find_element_by_id are deprecated (can be updated if needed)
